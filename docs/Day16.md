**continue** : 반복문의 다음 조건으로 넘어가게 한다.<br>
**exit** : 해당 쉘 프로그램을 완젼히 종료한다.<br>
**break** : 해당 반복문만 종료한다

### 함수

**선언**
```bash
함수명 () 
{

  return 
}
```

**호출**<br>
`함수명 [파라미터] [파마리터]`

**리턴값 처리**<br>
`변수=함수명`
